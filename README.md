# Кущай вкусно — Визитка ресторана грузинской кухни

Репозиторий содержит веб-приложение на Flask — многостраничную визитку ресторана грузинской кухни "Кущай вкусно".

**Возможности приложения:**
- Просмотр информации о ресторане (главная, "О нас")
- Меню с категориями блюд (аккордеон с картинками и описаниями)
- Отзывы гостей с общей средней оценкой и звёздочками
- Регистрация и авторизация пользователей
- Оставление, редактирование и удаление **своих** отзывов
- Админ-панель для управления меню и модерации отзывов
- Сортировка отзывов по дате и оценке (по возрастанию/убыванию)
- Удобный выбор изображений для блюд из папки `static/images/`
- Адаптивный дизайн на Bootstrap 5 с выдвижным боковым меню

**Особенности:**
- Безопасное хэширование паролей (Werkzeug)
- Разделение прав: обычные пользователи и администратор
- Администратор может добавлять/редактировать/удалять категории и блюда, удалять любые отзывы
- Полностью рабочие тесты на pytest (25 тестов)
- Согласие на обработку персональных данных (ссылка на PDF в футере)
- Современный минималистичный дизайн в грузинских тонах

**Структура проекта:**
- `app.py` — точка входа
- `config.py` — настройки (секретный ключ, параметры MySQL)
- `app/` — основной пакет приложения
  - `__init__.py` — создание приложения (factory pattern)
  - `routes.py` — все маршруты (основные страницы, аутентификация, админка)
  - `models.py` — модель пользователя (Flask-Login)
  - `forms.py` — формы WTForms
  - `templates/` — HTML-шаблоны (Jinja2)
    - `base.html` — общий шаблон с выдвижным меню и футером
    - `index.html`, `menu.html`, `about.html`, `reviews.html`
    - `auth/` — формы входа/регистрации/отзыва
    - `admin/` — админ-панель
- `static/`
  - `css/style.css` — кастомные стили
  - `images/` —  папка для фотографии блюд и интерьера
  - `privacy_policy.pdf` — согласие на обработку данных

**Установка и запуск**

1. Клонируйте репозиторий:
   ```bash
   git clone  https://github.com/Snoylex/ekzamen.git
   cd ekzamen

2. Создайте виртуальное окружение и активируйте его
- `python -m venv env`
- `env\Scripts\activate` # Windows
- `source venv/bin/activate` # macOS/Linux

3. Установите зависимости
- `pip install -r requirements.txt`

4. Импортируем файл БД
 - `base.sql в MySQL Workbench`

5. Запустите приложение
- `python app.py`

6. Перейдите по ссылке (http://127.0.0.1:5000)

**Тестирование**
1. Перейдите в корень проекта
2. Запустите тесты
- `python -m pytest tests/ -v`
